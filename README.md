# StorybookSearch

Simple Wikipedia Search App in React with Storybook for component managment

## Downloading dependencies
```bash
npm install
```

## Usage
```bash
// navigate to my-app directory
npm start
```

## Open Storybook
```bash
npm run storybook
```

## Testing
```bash
npm test

// generate coverage report
npm run test:ci

// open coverage report
npm run view:cov
```
