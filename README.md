# react-hooks-search

Simple Wikipedia search web app to explore React hooks with Jest and Storybook.

## Downloading dependencies
```bash
npm install
```

## Usage
```bash
// navigate to my-app directory
npm start
```

## Open Storybook
```bash
npm run storybook
```

## Testing
```bash
npm test

// generate coverage report
npm run test:ci

// open coverage report
npm run view:cov
```
